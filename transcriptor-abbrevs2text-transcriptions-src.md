

# Mansi abbreviations                               

We describe here how abbreviations are in Mansi are read out, e.g.
for text-to-speech systems.

* Root
For example:

 * s.:syntynyt # ;  
 * os.:omaa% sukua # ;  
 * v.:vuosi # ;  
 * v.:vuonna # ;  
 * esim.:esimerkki # ; 
 * esim.:esimerkiksi # ;
